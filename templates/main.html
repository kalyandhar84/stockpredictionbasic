× About Stock Price Prediction

Project Summary:

Based on current market trends, smart investments are critically important and the need of the hour. In order to invest smartly, we need to be aware of the market fluctuations, and trends and be able to use technology to reap the benefits to the fullest. Hence, we embark on this journey and decide to create an algorithm for predicting the stock prices based on historical trends with time series forecasting methodology. We have used the yahoo finance python library to get daily stocks data for analysis. TimeSeries forecasting is performed using LSTM and SARIMA models. In the next phase, we will use Python Flask for creating a web interface that will enable users to select the stocks they are interested in and provide forecasts for stock price movement.

Data Collection

A Python library called “yfinance” is used in order to import stock data. In order to keep our project simple, only 5 companies are considered. These 5 companies are Facebook (FB) Microsoft (MSFT) Amazon (AMZN) Netflix (NFLX) Apple (AAPL) Also, it is noted that some companies have split their stocks over time. Hence, instead of ingesting the daily stock price, the adjusted daily stock price is ingested in order to avoid the effect of stock splits.
